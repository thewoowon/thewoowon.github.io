# thewoowon.github.io
### 안녕하세요. 우원입니다.
### 캐스퍼오더라는 테이블링앱을 소개하는 웹 목업입니다. https://thewoowon.github.io에 배포되었습니다.

## 예상 기술 스택입니다.
### Front End 💁💁 React.js, HTML, SASS, CSS, JAVASCRIPT(ES6) ...+ Bootstrape
### Back End 💁💁 Node.js(express), MySQL
### State Management Tool 💁💁 REDUX 
### Version Tool 💁💁 Git
